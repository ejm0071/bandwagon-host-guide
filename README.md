## 先聊聊这家公司是谁

BandwagonHost，中文用户习惯叫它"搬瓦工"，因为拼音"banwagong"和品牌名发音实在太像了。它隶属于加拿大 IT7 Networks Inc.，2004 年就成立了，在 VPS 行业里算是老人了。

网站设计古朴，不花哨，不做大量广告，靠口碑在开发者和技术用户圈子里慢慢传开。核心优势就两个：硬件够扎实（企业级 RAID-10 SSD），网络线路优质（尤其是 CN2 GIA，中美之间的高速专线）。

官方支持支付宝和 PayPal，对国内用户友好。整个管理后台叫 KiwiVM，是他们自研的，功能很全，最好用的功能是**一键免费切换机房**——买了洛杉矶，觉得日本更快，点几下就迁移过去了，数据不丢。

## 搬瓦工是干什么的？适合我吗？

先问自己三个问题：

1. 你需要一台可以自己折腾的 Linux 服务器吗？
2. 你需要稳定的中美或中日网络连接？
3. 你有基本的 SSH 操作能力，或者愿意学？

三个都是，那搬瓦工基本上就是为你准备的。

搬瓦工属于**自管理型 VPS**，不帮你配置 cPanel，不提供一键建站，客服也不会帮你调 WordPress 插件。但如果你想要一台全权掌控的服务器，它给你 root 权限、PPP/VPN 支持、rDNS 配置，想怎么折腾就怎么折腾。
